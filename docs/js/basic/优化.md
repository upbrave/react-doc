### 移动端

#### passive

当 touchmove 事件中有耗时的操作时，导致页面滚动卡顿，可以使用 passive:true 选项避免卡顿

```js
window.onload = function () {
  var content = document.querySelector('.content');
  content.addEventListener(
    'touchmove',
    function (e) {
      var f = 0;
      document.title = f;
      for (let i = 0; i < 100000000; i++) {
        f += i;
      }
    },
    { passive: true },
  );
};
```
